Recovery Failsafe Kit

Store this recovery code in a password manager. It can unlock this encrypted .skye file if the passphrase is forgotten.

Encryption & Recovery Guide

1) Encrypt exports: Save As → select .skye → enable encryption checkbox → set passphrase.

2) Add a hint: Set a passphrase hint to help your team identify the right secret without exposing it.

3) Generate failsafe kit: Enable Recovery Failsafe Kit. Download and store in a separate secure vault.

4) Recovery path: On import, if passphrase fails you can unlock using the recovery code from the kit.

5) Safe practice: Never store passphrase and recovery kit in the same location. Use separate custody.

6) Rotation: Re-export encrypted files with fresh passphrase/recovery kit for quarterly key hygiene.
