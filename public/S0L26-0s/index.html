Build Rules

How To Add Or Remove Surfaces

  • Drop a new app or platform folder anywhere it should live in the public tree.
  • Add one entry to the local registry file for the new route, label, category, and summary.
  • Remove the entry from the same registry file when a surface should disappear from the hub.
  • Update the default workspace list if the new surface should open directly inside one of the three launcher frames.
