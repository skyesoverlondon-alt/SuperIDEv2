Workspace notes

Build scope

Included in this MVP: census management, chart editing, task tracking, SOAP drafting, handoff generation, calculators, and local JSON backup.

Not included: authentication, multi-user sync, audit trail, server persistence, role permissions, or EHR integration. Those would be next-lane work.

This is a stronger MVP skeleton than the original starter and keeps the single-file portability magic trick intact.
