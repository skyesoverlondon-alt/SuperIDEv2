# Doctor Ops Wave I

This ZIP contains 13 static browser apps focused on physician and clinic workflows.

What is inside:
- Intake & Triage Ops
- SOAP Note Studio
- Referral Router
- Prior Auth Tracker
- Lab Follow-up Board
- Medication Reconciliation Desk
- Imaging Results Commander
- Care Gap Radar
- Visit Prep Board
- Procedure Consent Vault
- Chronic Care Tracker
- Discharge Packet Builder
- Incident & Handoff Log

How to run:
1. Unzip the folder.
2. Open `index.html` in a browser.
3. Click any app card.

What each app supports:
- Local-first persistence with browser LocalStorage
- Create / update / clone / delete
- Search and filter
- Audit log
- JSON export / import
- CSV export
- Generated text summary export

Important:
- This is a static offline workflow suite.
- Use synthetic/demo data only in this build.
- It does not claim HIPAA certification, billing compliance, or legal sufficiency.
- It does not provide medical advice.

Entry point:
- `index.html`
