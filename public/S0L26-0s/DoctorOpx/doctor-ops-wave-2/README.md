# Doctor Ops Wave II

Open `index.html` in a browser after unzipping.

This bundle includes 13 doctor-focused static apps:
- Scheduling & Capacity Command
- No-Show Recovery Board
- Revenue Cycle Claims Tracker
- Denial Appeal Workbench
- Provider Credentialing Manager
- Staff Coverage Planner
- Preventive Screening Registry
- High-Risk Patient Watchlist
- Surgical Clearance Planner
- Vaccination Outreach Hub
- Device & DME Order Tracker
- Transitional Care Checkpoint
- Quality Measure Dashboard

Features in every app:
- local browser persistence with its own storage key
- create / update / clone / delete records
- search and status / priority filters
- JSON export / import
- CSV export
- generated summary box
- audit log
- synthetic seed data

This is a local demo scaffold. Use synthetic data only.
