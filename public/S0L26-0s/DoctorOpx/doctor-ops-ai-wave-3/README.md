# Doctor Ops AI Wave III

Open `index.html` after unzip.

This suite contains 13 offline-first doctor operations apps with:
- local browser persistence
- create / update / clone / delete workflows
- search and filters
- JSON import / export
- CSV export
- audit log
- embedded browser-side AI drafting panel

Important:
- Synthetic demo data only
- No live EHR, payer, or patient integration in this build
- Require clinician review before real-world use
