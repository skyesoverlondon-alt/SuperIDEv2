# Product Definition

Google Business Profile Rescue Platform helps local businesses and operators recover from GBP suspensions, ownership conflicts, and verification failures.

## Core user promise

- diagnose likely cause faster
- produce a coherent reinstatement package
- centralize evidence and notes
- track case progress across time

## Phase 1 user surfaces

- landing page
- login shell
- dashboard
- new case wizard
- case detail with AI generation actions
- billing summary page
- settings placeholder
- admin placeholder
